The class "ClassToRun" will be the public class of your code run, and should have main method. You can write methods you want to test in it as well as create private classes.
The maximum allowed time for the code compile/run is 1 minute. If your code doesn't compile or run in that time, the server will stop processing.
If the code compiles and there are no runtime errors, output to STDOUT will be displayed as output of the run.
